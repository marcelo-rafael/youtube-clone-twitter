- [x] TypeScript

## Starting Dev Environment

```bash
# Clone this repository
$ git clone https://github.com/marcelo-rafael/youtube-clone-twitter/
```

1. Run `npm install` or `yarn install`.<br />
2. Run `yarn start` and access `http://localhost:3000`.<br />
